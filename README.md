#  House Sales in King County, USA – Data Analysis with Python

This repository contains my **course project** completed as part of the **IBM Data Analysis with Python** course offered through Coursera. The project demonstrates an end-to-end data analysis workflow on the **House Sales in King County, USA** dataset using Python.

## 📖 Project Overview

The objective of this project is to analyze housing data to identify the key factors that influence house prices. The project covers the complete data analysis pipeline, from data preprocessing to building and evaluating predictive models.

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Analysis
- Feature Engineering
- Linear & Multiple Regression Modeling
- Model Evaluation

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📂 Dataset

- **Dataset:** House Sales in King County, USA
- **Type:** Real-world housing dataset

## 🎯 Learning Outcomes

Through this course project, I gained practical experience in:

- Cleaning and preparing real-world datasets
- Performing exploratory data analysis
- Creating meaningful visualizations
- Building and evaluating regression models
- Applying Python libraries for data analysis and machine learning

## 📌 Note

This project was completed as a **hands-on course project** for the **IBM Data Analysis with Python** course on Coursera. It is intended for learning purposes and demonstrates the practical application of fundamental data analysis techniques using Python.
